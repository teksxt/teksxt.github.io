Snake Classic — Privacy

This page summarises the data-safety declarations for Snake Classic as listed on Google Play. For the full privacy policy, see the Teksxt master privacy policy.

Google Play Data Safety

  • Data collected by the app: None.
  • Data collected by third parties: Advertising ID and approximate device data (via Google AdMob).
  • Data encrypted in transit: Yes (HTTPS).
  • Users can request data deletion: No Teksxt-held data to delete. Clear local data via Android Settings → Apps → Snake Classic → Storage → Clear data.

Permissions

  • Internet — to load ads.
  • Access Network State — to check connectivity before loading ads.
